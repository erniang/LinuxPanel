# LinuxPanel 更新日志

本文档记录了LinuxPanel项目的所有重要变更。

## [未发布]

### 新增
- 完整的前端界面，包括登录、仪表盘、网站管理、文件管理、数据库管理和应用商店
- 网站部署管理模块，支持Nginx配置和多PHP版本
- 数据库管理API，支持MySQL/MariaDB操作
- 文件管理功能，支持文件的浏览、上传、下载和编辑
- 系统监控功能，实时显示CPU、内存和磁盘使用情况
- 应用商店基础功能，支持应用的安装和管理
- 一键安装脚本，支持多种Linux发行版

### 优化
- 改进了安装过程，增加了环境检测和依赖自动安装
- 优化了前端界面响应速度和用户体验

### 修复
- 修复了循环导入问题
- 修复了配置文件解析错误
