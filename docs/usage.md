## 登录面板

1. 打开浏览器，访问您的服务器IP地址，例如：`http://YOUR_SERVER_IP`
2. 您将看到登录页面，输入用户名和密码
   - 默认用户名：`admin`
   - 默认密码：`admin`
3. 点击"登录"按钮进入管理界面
4. 首次登录后，系统会提示您修改默认密码

> **安全提示**：请务必在首次登录后立即修改默认密码，以防止未授权访问。

## 系统概览

系统概览页面显示服务器的基本信息和资源使用情况：

- **资源监控**：实时显示CPU、内存、硬盘使用率
- **系统信息**：操作系统版本、内核版本、主机名等基本信息
- **运行时间**：系统运行时间和负载情况
- **资源图表**：CPU和内存使用率的实时图表

**刷新数据**：页面默认每30秒自动刷新一次，您也可以点击右上角的刷新按钮手动刷新数据。

## 网站管理

网站管理模块允许您轻松创建和管理多个网站。

### 创建网站

1. 在网站管理页面，点击"创建网站"按钮
2. 填写以下信息：
   - **网站名称**：给您的网站起一个易于识别的名称
   - **域名**：网站的域名（例如：example.com）
   - **网站路径**：网站文件存放的目录路径
   - **PHP版本**：选择适合您网站的PHP版本（如果需要）
   - **启用SSL**：是否为网站启用HTTPS
3. 点击"确认"按钮创建网站

### 网站设置

对于每个网站，您可以管理以下设置：

1. 点击网站列表中的"设置"按钮
2. 在弹出的对话框中，您可以在不同选项卡中修改网站配置：
   - **基本设置**：修改网站名称、域名、文件路径等
   - **SSL设置**：配置SSL证书
   - **数据库设置**：管理网站关联的数据库

### SSL证书管理

您可以通过以下方式为网站配置SSL：

1. 在网站设置中，切换到"SSL设置"选项卡
2. 选择SSL类型：
   - **Let's Encrypt自动申请**：系统自动申请免费的Let's Encrypt证书（需要您的域名已正确解析到服务器）
   - **自定义证书**：上传您自己的SSL证书和私钥
3. 点击"保存"应用SSL设置

> **注意**：使用Let's Encrypt时，请确保您的域名已正确解析到服务器IP，否则证书申请将失败。

### 网站操作

网站列表提供了以下操作按钮：

- **启动/停止**：开启或关闭网站
- **重启**：重新加载网站配置
- **删除**：永久删除网站及其配置（请谨慎操作）

## 文件管理

文件管理器提供了图形化界面来浏览和操作服务器上的文件。

### 浏览文件

1. 在左侧导航栏点击"文件管理"进入文件管理器
2. 双击文件夹进入，点击路径导航栏可以快速切换目录
3. 使用搜索框可以在当前目录中查找文件
4. 点击列标题可以按名称、大小、类型或修改时间排序

### 文件操作

文件管理器支持以下基本操作：

- **上传文件**：点击工具栏中的"上传"按钮
- **下载文件**：选中文件后点击"下载"，或右键菜单选择"下载"
- **新建文件/文件夹**：点击对应的"新建文件"或"新建文件夹"按钮
- **剪切/复制/粘贴**：通过工具栏或右键菜单操作
- **重命名**：选中文件，点击"重命名"或按F2键
- **删除**：选中文件，点击"删除"或按Delete键

### 文件编辑器

对于文本文件，您可以使用内置的编辑器进行在线编辑：

1. 点击文本文件打开编辑器
2. 编辑器支持语法高亮和代码折叠
3. 修改完成后点击"保存"按钮保存更改
4. 点击"关闭"按钮返回文件列表

### 文件权限管理

您可以修改文件和文件夹的权限：

1. 选中文件，右键选择"属性"或点击工具栏的"属性"按钮
2. 在属性对话框中，切换到"权限"选项卡
3. 修改文件的所有者、所属组和权限设置
4. 点击"应用"保存更改

## 数据库管理

数据库管理模块允许您创建和管理MySQL/MariaDB数据库。

### 创建数据库

1. 在数据库管理页面，点击"创建数据库"按钮
2. 填写以下信息：
   - **数据库名称**：新数据库的名称
   - **字符集**：选择数据库字符集，通常使用utf8mb4
   - **排序规则**：选择适合您语言的排序规则
3. 点击"确认"创建数据库

### 数据库用户管理

创建和管理数据库用户：

1. 在数据库管理页面，切换到"用户管理"选项卡
2. 点击"创建用户"按钮
3. 填写用户信息：
   - **用户名**：数据库用户名
   - **密码**：强密码
   - **主机**：允许连接的主机（localhost或IP地址）
   - **权限**：选择要授予的数据库和权限
4. 点击"创建"按钮

修改现有用户的权限或密码：

1. 在用户列表中找到要修改的用户
2. 点击"权限"按钮修改权限
3. 点击"修改密码"按钮更改用户密码

### 备份与恢复

备份数据库：

1. 在数据库列表中，找到要备份的数据库
2. 点击"备份"按钮
3. 系统会自动创建数据库备份并提供下载链接

恢复数据库：

1. 在数据库管理页面，点击"恢复数据库"按钮
2. 选择数据库和备份文件
3. 点击"开始恢复"，系统将从备份文件中恢复数据库

## 应用商店

应用商店提供了一键安装常用Web应用的功能。

### 安装应用

1. 在左侧导航栏点击"应用商店"
2. 浏览可用应用列表或使用搜索框查找特定应用
3. 点击应用卡片查看详细信息
4. 点击"安装"按钮
5. 填写安装配置信息：
   - **安装目录**：应用安装的目标目录
   - **域名**：应用访问的域名
   - **数据库配置**：是否为应用创建新数据库
   - **管理员信息**：应用管理员的用户名和密码
6. 点击"开始安装"，系统将自动安装应用

### 管理应用

已安装的应用可以在"已安装"选项卡中管理：

- **访问**：打开应用网站
- **更新**：更新应用到最新版本（如果有可用更新）
- **设置**：修改应用配置
- **卸载**：卸载应用

### 应用源管理

LinuxPanel支持自定义应用源：

1. 在应用商店页面，点击"应用源"按钮
2. 查看当前配置的应用源列表
3. 点击"添加应用源"按钮添加新的源
4. 填写源信息：
   - **名称**：应用源名称
   - **URL**：应用源地址
   - **优先级**：数字越小优先级越高
5. 点击"添加"按钮保存

## 系统设置

系统设置模块允许您配置面板的全局设置。

### 用户管理

管理面板用户：

1. 在系统设置中，切换到"用户管理"选项卡
2. 查看现有用户列表
3. 点击"添加用户"创建新用户
4. 设置用户信息：
   - **用户名**：登录用户名
   - **密码**：登录密码
   - **邮箱**：用户邮箱地址
   - **角色**：选择用户角色（管理员或普通用户）
5. 点击"创建"按钮

### 安全设置

配置面板的安全选项：

1. 在系统设置中，切换到"安全设置"选项卡
2. 配置以下选项：
   - **会话超时时间**：自动注销前的不活动时间
   - **失败登录限制**：防暴力破解的登录尝试限制
   - **面板访问IP限制**：限制可访问面板的IP地址
   - **开启二步验证**：为用户启用两因素认证
3. 点击"保存"应用更改

### 常规设置

配置面板的基本设置：

1. 在系统设置中，切换到"常规设置"选项卡
2. 配置以下选项：
   - **面板名称**：自定义面板显示的名称
   - **主题设置**：选择面板主题（亮色/暗色）
   - **语言设置**：设置面板界面语言
   - **自动更新**：配置面板自动更新选项
3. 点击"保存"应用更改

## 故障排除和提示

- **操作失败**：大多数操作失败会有明确的错误提示，请根据提示解决问题
- **页面未响应**：如果页面长时间未响应，尝试刷新浏览器
- **会话过期**：如果您长时间未操作，会话可能会过期，需要重新登录
- **权限问题**：某些文件操作可能因权限不足而失败，请确保面板有足够的权限

## 最佳实践

1. **定期备份**：定期备份重要的网站数据和数据库
2. **密码安全**：使用强密码，定期更换密码
3. **保持更新**：定期更新面板和已安装的应用
4. **监控资源**：关注系统资源使用情况，避免资源耗尽
5. **权限控制**：遵循最小权限原则，不要给文件或用户过高的权限
